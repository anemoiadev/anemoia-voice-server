# Anemoia Backend (Voice & Chat Server)

Это серверная часть (Ядро) моего пет-проекта Anemoia - кастомного браузера/клиента с упором на приватность и риал-тайм взаимодействие.

**Стек технологий:**
* **Node.js** & **Express** (REST API)
* **Socket.io** (Коммутатор реального времени, WebSockets)
* **MongoDB / Mongoose** (Хранение профилей)

**Почему тестировщику полезно писать сервер?**
Разработка этого проекта позволила мне глубоко разобраться в клиент-серверной архитектуре. Я понимаю, как формируются JSON-ответы, как перехватывать и отлаживать Socket-соединения, как работают таймауты, токены авторизации и защита баз данных. Этот опыт помогает мне при ручном тестировании API находить архитектурные и логические уязвимости еще до того, как они попадут на клиент.
